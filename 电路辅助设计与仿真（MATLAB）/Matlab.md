### 3.1-1 基本二维曲线绘制
    plot(y)
    plot(x,y)
    
    1.plot(y)
    （1）y是向量的时候，假设y=[y1, y2, y3]，则plot(y)会将y的元素作为y轴的坐标，x轴的坐标默认为1、2、3. 结果：画出（1, y1）、（2, y2）、（3, y3）三个点，并用线段连接起来，形成一条曲线。
    （2）y是矩阵的时候，y每一列视作一个向量，plot(y)会将每一列的元素作为y轴的坐标，x轴的坐标默认为1、2、3... 结果：画出多条曲线，每条曲线对应y的一列。
    （3）y是复数向量的时候，每一个复数为a+bi，，则画出点（a, b），并用线段连接起来，形成一条曲线。

    2.plot(x, y)
    （1）x和y都是向量的时候，plot(x, y)会将x的元素作为x轴的坐标，y的元素作为y轴的坐标。结果：画出点（x1, y1）、（x2, y2）、（x3, y3）...，并用线段连接起来，形成一条曲线。
    （2）x是向量，y是矩阵的时候，plot(x, y)会将x的元素作为x轴的坐标，y每一列视作一个向量，plot(x, y)会将每一列的元素作为y轴的坐标。结果：画出多条曲线，每条曲线对应y的一列。
    （3）x和y都是矩阵的时候，plot(x, y)会将x的每一列视作一个向量，y的每一列视作一个向量，plot(x, y)会将x的每一列的元素作为x轴的坐标，y的每一列的元素作为y轴的坐标。结果：画出多条曲线，每条曲线对应x的一列和y的一列。

    plot(x,[y1;y2])

    多项式的四则运算
    多项式求导、求根和求值
    多项式拟合
    线性微分方程的解
    
    1.多项式的四则运算：
    多项式的表示：
    用一个向量来表示多项式，向量的每个元素对应多项式的一个系数，按照降幂排列。例如，多项式2x^3 + 3x^2 + 5可以表示为向量[2, 3, 0, 5]。

    多项式的计算：
    多项式的加法：C = A + B; % 多项式A和B的加法，结果为多项式C
    多项式的减法：C = A - B; % 多项式A和B的减法，结果为多项式C
    多项式的乘法：C = conv(A, B); % 多项式A和B的乘法，结果为多项式C
    多项式的除法：[Q, R] = deconv(A, B); % 多项式A除以B，结果为商多项式Q和余数多项式R

    2.多项式求导、求根和求值：
    多项式求导：dA = polyder(A); % 计算多项式A的导数，结果为多项式dA
    多项式求根：r = roots(A); % 计算多项式A的根，结果为向量r，包含多项式A的所有根
    多项式求值：y = polyval(A, x); % 计算多项式A在x处的值，结果为y

    3.多项式拟合：
    p = polyfit(x, y, n); % 对数据点(x, y)进行n次多项式拟合，返回拟合多项式的系数向量p

    首先利用**拉普拉斯变换**，将线性常微分方程转化为**代数方程**。  
    其响应的拉普拉斯域表达式通常可写为：

    $$
    Y(s)=\frac{B(s)}{A(s)}
    $$

    其中：

    - $B(s)$：分子多项式
    - $A(s)$：分母多项式

    一般假设 **分母阶数高于分子阶数**，这样便于进行部分分式展开。
